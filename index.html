WSOLO SPECIFICATION

Ввод и Вывод

Console.New.Text = "Текст" — Стандартный вывод.
Read(prompt, settings) — Получение данных.
Аргумент settings позволяет применять Python-методы к вводу через встроенный eval.

Директива Pass

use pass * [Module] — Глобальный проброс. Убирает необходимость указывать имя модуля перед его командами (работает для Console и любых других подключаемых библиотек).

Система Use

use <System> — Математика и Zona.
use <gameengine> — Визуальный движок.
use <time> — Модуль управления временем.
use <videoplayer> — Работа с видео (cv2).
use <pygame> — Библиотека pygame.

Управление кодом

tab / line / space — Формирование структуры файла work.py.
workwrite [code] — Прямая запись строки в исполняемый файл.
time [sec] — Пауза (sleep).
exit — Прерывание работы транслятора.
